GUIs (Graphical User Interfaces)

Interfaces that allow users to interact with a computer using pictures,
buttons, windows, icons, and mouse.
Examples:

.Windows OS
.Android apps
.Browsers (Chrome, Firefox)
.Games
.Calculator app

==Features:
.Easy to use
.Mouse/Touchscreen-based
.Visual elements (buttons, menus)
.Beginner-friendly
